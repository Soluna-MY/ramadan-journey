# Ramadan Journey

An open-source application to help Muslims track their prayers, read daily hadiths, and monitor their Quran reading during the holy month of Ramadan.

## Features

*   **Daily Hadith**: Get a new hadith every day to inspire and guide you.
*   **Prayer Times**: View daily prayer times based on your location.
*   **Quran Tracker**: Keep track of your Quran reading progress.

## Tech Stack

This project is built with:

*   Vite
*   TypeScript
*   React
*   shadcn-ui
*   Tailwind CSS
*   Vercel Analytics

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js and npm
    *   [Install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your-username/ramadan-journey.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```

## Deployment

You can deploy this project to any static site hosting service. A good option is [Vercel](https://vercel.com/).
